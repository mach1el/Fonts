# Installtion

*  With per user

        mkdir ~/.fonts
        cd ~/.fonts
        git clone https://github.com/t7hm1/Fonts.git
        fc-cache -vf ~/.fonts
        
*  With wide system

        cd /usr/share/icons
        sudo -s
        git clone https://github.com/t7hm1/Fonts.git
        sudo fc-cache -fv
